カスタム時間割メーカー 📅

ブラウザで動く、シンプルで高機能な週間時間割作成ツールです。
お子様向けの「かわいい」デザインから、掲示用の「シンプル」なデザインまで、ボタンひとつで切り替えられます。
A4横向き印刷に最適化されており、PDFとしての保存も可能です。

✨ 特徴

4つの選べるデザインテーマ

⭐ ポップ: 低学年のお子様向け。元気でかわいい色使い。

🍃 自然: カフェ風の落ち着いた手書き風デザイン。

⚡ クール: 高学年向け。寒色系ですっきりとしたデザイン。

💼 基本: インク節約にもなる、シンプルで見やすいデザイン。

2つの表示モード

児童用: 科目名を大きく表示。ひらがな・漢字も見やすく。

詳細（保護者用）: 時間、教室、先生名などの詳細情報を表示。

かんたん編集

セルをクリックするだけで、科目・先生・場所を入力可能。

カレンダーから期間（開始日）を選ぶだけで、日付を自動入力。

カスタマイズ自由

科目の色や名前を自由に追加・変更できます。

インストール不要

HTMLファイルひとつで動作します。ネット環境がなくても使えます（初回読み込みを除く）。

🚀 使い方

1. 起動方法

ダウンロードした timetable_maker.html をダブルクリックして、ChromeやEdge、Safariなどのブラウザで開いてください。

2. 基本的な操作

セルの編集: 時間割の枠をクリックすると編集画面が開きます。科目を選んだり、先生の名前を入力して「保存」を押してください。

期間の設定: 左側の設定パネルにあるカレンダーアイコンをクリックして日付を選ぶと、タイトル下の期間が自動で更新されます。

デザイン変更: 画面上部のボタンで、お好みのテーマやモードに切り替えられます。

3. 保存と印刷

画面右上の 「印刷 / PDF出力」 ボタンを押してください。

プリンターの設定画面が開きます。

紙で印刷する場合: プリンターを選択して印刷します（「横向き」推奨）。

データで保存する場合: プリンターの選択肢から「PDFとして保存」を選んでください。

4. データの保存について

入力したデータは、お使いのブラウザ（ローカルストレージ）に自動的に保存されます。
次回同じブラウザでファイルを開くと、前回の続きから編集できます。

⚠️ 注意事項

ブラウザのキャッシュ（履歴）を削除すると、保存された時間割データも消える場合があります。

推奨印刷サイズは A4（横向き） です。
